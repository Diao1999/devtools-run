# ⚡ DevForge

Free online developer tools. Fast, private, no tracking.

All processing happens in your browser — no data is sent to any server.

## Tools

- **JSON Formatter** — Format, minify, and validate JSON
- **Base64** — Encode and decode Base64 strings
- **URL Encode/Decode** — Encode and decode URL components
- **Timestamp** — Convert between Unix timestamps and dates
- **UUID Generator** — Generate random UUIDs (v4)
- **Hash Generator** — SHA-1, SHA-256, SHA-512 hashes
- **Regex Tester** — Test regular expressions with real-time matching
- **JWT Decoder** — Decode and inspect JSON Web Tokens
- **Color Converter** — Convert between HEX, RGB, and HSL
- **Markdown Preview** — Preview Markdown with live rendering

## Features

- 🔒 100% client-side — your data never leaves your browser
- ⚡ Instant — no loading, no frameworks, no build step
- 🎨 Dark theme — easy on the eyes
- 📱 Responsive — works on mobile
- 🚀 ~20KB total — lighter than most favicons
